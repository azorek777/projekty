MixR - programmable console

About MixR

MixR is a project that is completely free and allows the user to create their own device that will look as they want. MixR is to be a response to much more expensive counterparts such as StreamDeck or Razer Stream Controller. As manufacturers, we provide our own software that allows for convenient operation of the equipment

General Functions

  1. Set the volume for specific apps

  2. Set the microphone volume

  3. Assigning functions to buttons:

      using system shortcuts
      running specified programs
      making macros
